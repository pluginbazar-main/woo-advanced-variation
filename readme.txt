	Advanced multifunctional product variation swatches for WooCommerce with Colors, Images and Buttons.

== Description ==

Variation Swatches is an extension for WooCommerce Plugin. It will provide you ability to allow your customers to select product variation easily. Checkout some awesome features:

Variation attributes can be different types like – Images, Colors, Buttons where WooCommerce provide only a simple type.

##Features

* Select choices from Product archive, No need to go single product page.
* Beautiful Presentation in Single Product Page.
* Show prices upon selection.
* Change any content or style like – Color, Background color, Font size, Font style, Font weight and many more.
* Show your payment gateways to increase Trust value to your customers.
* Filters and Actions available to extend for Developers.
* Responsive design
* High performance on website load speed
* Security ensured
* Guarantee of regular updates
